#Enable-iPad-LTE
为 iPad 打开中国联通网络的4G开关（无需越狱）

简介
---
  由于众所周知的原因，蜂窝版本iPad的是没有LTE开关的（特定型号除外），Apple为每个运营商配置了不同的文件，而iPad在插入Sim卡后会检查其运营商并使用相应的配置文件，以此决定是否开放LTE开关。在默认情况下，中国联通的运营商文件并没有开放LTE开关。这里提供的描述文件将更新中国联通的运营商文件以此开放LTE开关
